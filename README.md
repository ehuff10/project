# Airline Flight Delay Dashboard

This project analyzes U.S. airline flight delay data and presents insights through an interactive dashboard.
The goal is to understand operational delay patterns by airline, airport, and time, and to predict delay risk
using historical flight data.

## Project Structure

- data/raw: raw flight data (not committed)
- data/processed: cleaned datasets
- notebooks: exploratory analysis and modeling
- app: Streamlit dashboard application

## Key Features

- Exploratory data analysis of flight delays
- Delay risk classification model
- Interactive dashboard with simulated live updates

## Tech Stack

- Python
- Pandas / NumPy
- Scikit-learn
- Streamlit
